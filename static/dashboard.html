Our framework

Creating applications running on a quantum computer is currently not yet standardised. There are no end-to-end systems to help web developers exploring quantum world We propose a set of methods that allows to:

- create and deploy web applications that use a quantum computer as back-end
- open to an end-to-end solution with all the needed structures for web hybrid classical-quantum computing

This proposed set of methods could be implemented using various technologies.


The prototype we created based on this system. It is a set of web applications that:

- take a set of inputs from a user interface
- sends it to the back-end
- performs some operations with classical and quantum computing o send back the output to the front-end to be visualised

It relies on an approach to quantum web applications oriented to real business (alternative to Jupyter Notebooks) and it is easily deployable on cloud (it is cloud native)
